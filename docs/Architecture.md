# JBI Workshop Manager

## Vision

JBI Workshop Manager is a modern Windows desktop ERP and workshop management system developed for John Berends Implements.

The software is built using:

- C#
- .NET 8
- WPF
- MVVM
- SQLite
- Entity Framework Core
- Dependency Injection
- Serilog
- Git & GitHub

## Architecture Goals

- Maintainable
- Modular
- Fast
- Testable
- Professional
- Scalable
